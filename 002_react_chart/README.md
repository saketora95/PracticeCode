# 目標
1. 每 10 秒讀取指定網址，並將讀取結果以折線圖表示。
2. 橫軸區間不超過 10 分鐘之外，可自由設置。
3. 縱軸以取得數值的最大值 + 20。
4. 每次更新時會顯示平均線。
